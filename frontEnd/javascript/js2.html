# break 和 continue 语句

break 和 continue 语句用于精确的控制代码的执行。

  1. break会立即退出循环，并强制继续执行循环后续的语句；
  2. continue也是立即退出循环，不同的是，退出循环后会从循环的顶部继续执行；

# break 执行次数，代码如下：

    var num = 0;
    for(var i=1; i < 10; i++) {
        if (i % 5 == 0) {
            break;
        }
        num++;
    }
    alert(num);  //4

# continue的执行次数，代码如下

    var num = 0;
    for(var i=1; i < 10; i++) {
        if (i % 5 == 0) {
            continue;
        }
        num++;
    }
    alert(num);  //8

# break结合label使用，代码如下：

    var num = 0;
    outermost:
    for(var i=0; i < 10; i++) {
        for(var j=0; j < 10; j++) {
            if(i == 5 && j == 5) {
                break outermost;
            }
            num++;
        }
    }
    alert(num); //55

# continue结合label使用，代码如下：

    var num = 0;
    outermost:
    for(var i=0; i < 10; i++) {
        for(var j=0; j < 10; j++) {
            if(i == 5 && j == 5) {
                continue outermost;
            }
            num++;
        }
    }
    alert(num); //95

# 函数传递参数

按值传递：变量复制基本类型值时，操作副本不会影响原变量, 基本类型存储在栈内存中。 按引用传递：变量复制引用类型值时，操作副本会影响原变量，因为，引用的值是通一个对象。 引用类型存储在堆内存中

函数的参数是按值传递还是按引用传递？
